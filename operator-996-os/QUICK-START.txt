## Troubleshooting (Quick Fixes)

| Problem | Solution |
|---------|----------|
| Port 3000/8000 in use | `lsof -i :3000` â†’ `kill -9 <PID>` OR change port in docker-compose |
| Backend won't start | `pip install -r requirements.txt` OR check `docker logs operator996-backend` |
| Frontend blank | Refresh page, check network tab in browser dev tools |
| WebSocket error | Ensure backend is running, check firewall rules |
| "Cannot find module" | Run `npm install` in frontend folder |
| Permission denied | Run with `sudo` OR check Docker daemon running |
